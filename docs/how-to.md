# How to use Groundwork

> Pipeline backend is not live yet. This doc reflects what's built.

---

## `/start`

1. Pick a market from the grid.
2. Type an idea and hit **dump** (or `Cmd+Enter`) to save it. Repeat.
3. Ideas persist across sessions — close the tab, they're still there.
4. Select a market, then hit **Run pipeline** when it's wired.

---

## Output

When the pipeline finishes, you get `context.md` + `brand.md`.  
Drop them into the boilerplate and run `/setup`.
